# MGLOC Project

This project implements and evaluates the M-GLOC bandit algorithm.

## Structure

- `main.py`: The main entry point to run experiments.
- `src/mgloc/`: Contains the core source code.
- `src/mgloc/algorithms/`: Bandit algorithm implementations.
- `src/mgloc/environment.py`: The simulation environment.
- `src/mgloc/experiment.py`: Experiment orchestration logic.
- `figures/`: Output for generated plots.
- `notebooks/`: Jupyter notebooks for exploration.

## Usage

To run the main experiment:
```zsh
python main.py
```
